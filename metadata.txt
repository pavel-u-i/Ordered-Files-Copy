about=Этот инструмент копирует файлы, ссылки на которые содержатся в атрибутах объектов слоя, с возможностью иерархичной группировки файлов в соответствии с другими атрибутами объектов слоя. Допустимо использование в отношении объектов слоя, для которых последовательно через разделитель указаны пути для нескольких файлов (без дополнительных знаков выделения, например кавычек). Инструмент имеет графический интерфейс выбора поля атрибутов, содержащего ссылку; интерфейс ввода используемого разделителя, а также интерфейс выбора типа копирования файлов: с иерархичной группировкой или без таковой. В случае группировки активизируется интерфейс создания иерархичной структуры путем выбора полей атрибутов слоя с учетом последовательности их перечисления.| This tool copies files referenced in layer object attributes, with the option to group files hierarchically according to other layer object attributes. It can be used for layer objects for which paths to multiple files are specified sequentially using a separator (without additional delimiters, such as quotation marks). The tool has a graphical interface for selecting the attribute field containing the link, an interface for entering the separator used, and an interface for selecting the type of file copying: with or without hierarchical grouping. In the case of grouping, the interface for creating a hierarchical structure is activated by selecting layer attribute fields, taking into account the sequence in which they are listed.
